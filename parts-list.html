  • Latency~1ms

  • HDR InfiniBand Switch (36-port)$8,000-12,000

  • HDR Cables$500-1,000

🔧 Networking Recommendations

  • 2-4 node clusters: Start with Gigabit Ethernet, upgrade to 10GbE if needed
  • 4-8 node clusters: 10 Gigabit Ethernet provides good price/performance
  • 8+ node clusters: Consider 25GbE RoCE or used InfiniBand equipment
  • Budget tip: Used ConnectX-3/4 InfiniBand cards cost $50-200 and provide huge improvements
